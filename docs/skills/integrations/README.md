# Orca Coder — integration wizard docs

Short setup notes for each **Integrations** wizard entry. Full upstream skills live under `~/.claude/skills/` when installed (especially Obsidian).

## Skill sources matrix

The **full GitHub curation** (repo links, license notes, maintenance, browser vs API fit, in-repo vs external-only) lives in **[CURATED_SKILL_SOURCES.md](CURATED_SKILL_SOURCES.md)**.

In the app, open **Integrations…** (title bar) or **Settings → Integrations**, then use the **Skill sources** tab in the wizard to read the same document without leaving Orca.

## Opening the wizard

- **Title bar:** **Integrations…**
- **Settings:** **Integrations** → **All integrations…** or **Add Obsidian** (one-click vault brain)
