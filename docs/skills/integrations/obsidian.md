# Obsidian

**Skills (install via Claude Code / skills path):** [kepano/obsidian-skills](https://github.com/kepano/obsidian-skills) — e.g. `obsidian-markdown`, `obsidian-cli`, `json-canvas`, `obsidian-bases`. In Orca, **`read_file`** the matching `SKILL.md` before bulk-editing vault files.

**Workspace:** Open your vault folder (or a parent folder) as the Orca **workspace** so paths line up with tools.

**Orca:** No separate “Obsidian app” tile—use **editor** + **browser** + orchestrator file tools on the vault on disk.
